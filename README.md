# Personal Diary Website

A responsive Personal Diary application built using HTML, CSS, JavaScript and Firebase.

Features:

• User Registration

• User Login

• Forgot Password

• Search Notes

• Date Wise Notes

• Dark Mode

• Auto Save

• Logout

• Responsive Design

Technologies Used:

• HTML

• CSS

• JavaScript

• Firebase Authentication

• Firebase Firestore
